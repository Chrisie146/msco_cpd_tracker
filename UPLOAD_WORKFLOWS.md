# CPD Activity Upload - Two Workflows

## Workflow 1: Manual Entry with File Upload (No AI Needed) ‚úÖ

**You can ALWAYS upload documents without AI!**

### Steps:
1. **Go to Reflection Phase** (Phase 3 - the purple button)
2. **Click "Add CPD Activity"** button
3. **Fill in the form manually**:
   - Date: When the activity happened
   - Activity Name: What you did (e.g., "SAICA Advanced Tax Webinar")
   - Activity Type: Choose from dropdown (Formal, Informal, Compulsory, Webinar, Conference, Reading, Other)
   - Competency Area: What skill area (e.g., "Taxation")
   - CPD Hours: How many hours (e.g., 2)
   - Provider: Organization (e.g., "SAICA")
   - Description: Brief summary
   - Outcome/Reflection: What you learned

4. **Upload Evidence** (Optional but recommended):
   - Click the dashed upload box
   - Select file (PDF, JPG, PNG, DOC)
   - Can upload multiple files
   - Click the Trash icon to remove any file

5. **Click "Add Activity"** to save everything with your files

### That's it! ‚úÖ
- Activity saved with all details
- Evidence files stored with it
- Can download files anytime from the activity list

## Workflow 2: AI-Assisted (Optional Shortcut)

**If you want AI to auto-fill the form:**

1. Ensure VITE_CLAUDE_API_KEY is configured
2. Go to Reflection Phase
